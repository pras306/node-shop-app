## How to use

Clone the project

```bash
    git clone https://github.com/pras306/node-shop-app.git
```

Go to project directory

```bash
    cd node-shop-app
```

Install the dependencies

```bash
    npm install
```

Start the server

```bash
    npm start
```

Start development server

```bash
    npm run dev
```
